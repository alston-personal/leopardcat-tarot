# Card Format

## Reference Tarot Format

For the first visual draft, the card uses the common tarot proportion:

- physical reference: `2.75 x 4.75 in`
- ratio: `11 : 19`
- decimal ratio: `1 : 1.727`

This sits close to many standard tarot decks and gives enough vertical space for:

- title band
- central illustration
- bottom subtitle / number zone

## Draft Canvas

The current draft render uses:

- `1400 x 2420 px`

This preserves the tarot-like proportion while giving enough resolution for review and iteration.

## Layout Zones

1. Top title band
2. Main illustration frame
3. Bottom card number and subtitle
4. Outer border and inner line frame

## Style Direction For Draft 01

- engraved / illustrated poster feeling
- dark forest palette
- warm road-light contrast
- conservation mood rather than glossy fantasy
